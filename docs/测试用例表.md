## 一、制冷功能测试

| 编号 | 测试项 | 测试步骤 | 预期结果 | 实际结果 | 状态 |
|------|--------|----------|----------|----------|------|
| C-01 | PID初始化 | 上电后检查 `g_cooling_pid` 状态 | `enabled=0, output=0, target_temp=100(10°C)` | | ☐ |
| C-02 | CCP0 PWM初始化 | 上电后测量P30引脚 | P30输出1kHz PWM，占空比0% | | ☐ |
| C-03 | PID使能 | 发送启动指令 `01 01` | `g_cooling_pid.enabled=1` | | ☐ |
| C-04 | PID禁用 | 发送停止指令 `01 00` | `enabled=0, output=0, Cooling_Off()` | | ☐ |
| C-05 | NTC温度读取 | 检查 `ADC_Read(9)` 返回值 | 返回非零ADC值（室温约1300~2000） | | ☐ |
| C-06 | ADC_DrvScan工作 | 主循环中调用 `ADC_DrvScan()` | `_adcResult[9]` 随温度变化更新 | | ☐ |
| C-07 | 温度转换 | `NTC_ADC_ToTemp(1500)` | 返回值在150~250之间（15~25°C） | | ☐ |
| C-08 | 制冷档位1 | 发送指令 `0C 01` | `target_temp=200(20°C)` | | ☐ |
| C-09 | 制冷档位2 | 发送指令 `0C 02` | `target_temp=170(17°C)` | | ☐ |
| C-10 | 制冷档位3 | 发送指令 `0C 03` | `target_temp=130(13°C)` | | ☐ |
| C-11 | 制冷档位4 | 发送指令 `0C 04` | `target_temp=80(8°C)` | | ☐ |
| C-12 | 制冷档位5 | 发送指令 `0C 05` | `target_temp=50(5°C)` | | ☐ |
| C-13 | 档位越界保护 | 发送指令 `0C 00` 或 `0C 06` | 目标温度不变，不崩溃 | | ☐ |
| C-14 | PID输出响应 | 启动后等待2秒，观察PWM | 当温度>目标时，P30占空比增大 | | ☐ |
| C-15 | PID积分限幅 | 持续运行5分钟 | `integral` 不超过 `PID_INTEGRAL_MAX(5000)` | | ☐ |
| C-16 | 制冷关闭 | 发送停机指令后测量P30 | PWM占空比归零，CCP0停止 | | ☐ |
| C-17 | 蓝牙断开自动关制冷 | 拔掉蓝牙模块模拟断开 | `enabled=0, Cooling_Off()` 自动执行 | | ☐ |

## 二、SN74HC21D半桥控制测试

| 编号 | 测试项 | 测试步骤 | 预期结果 | 实际结果 | 状态 |
|------|--------|----------|----------|----------|------|
| E-01 | GPIO初始化验证 | 上电后测量P10/P12/P31/P40 | 四个GPIO均为输出低电平 | | ☐ |
| E-02 | EPWM初始化验证 | 上电后测量P00和P06 | P00(EPWM2)和P06(EPWM3)均输出3kHz PWM | | ☐ |
| E-03 | 上半桥开启 | 调用 `SN74HC21D_OpenUpper()` | P31/P10使能拉高，P06(EPWM3)输出PWM波形 | | ☐ |
| E-04 | 下半桥开启 | 调用 `SN74HC21D_OpenLower()` | P12/P40使能拉高，P00(EPWM2)输出PWM波形 | | ☐ |
| E-05 | 死区保护 | 先开启上半桥，再快速开启下半桥 | 上下桥切换时有死区延时，示波器确认无重叠导通 | | ☐ |
| E-06 | 占空比设置 | 调用 `SN74HC21D_SetDutyUpper(50)` | 示波器测量P06(EPWM3)占空比为50% | | ☐ |
| E-07 | 正弦波生成 | 调用 `SN74HC21D_SineWaveEnable(100)` | 示波器测量变压器输出为100Hz阶梯正弦波 | | ☐ |
| E-08 | 正弦波停止 | 调用 `SN74HC21D_StopAll()` | 所有GPIO输出低，EPWM停止，变压器无输出 | | ☐ |
| E-09 | 半桥互斥 | 上半桥工作时调用 `SN74HC21D_OpenLower()` | 上半桥先关闭(含死区延时)，下半桥再开启 | | ☐ |
| E-10 | EPWM中断标志清除 | 启动正弦波后检查 `EPWM_IRQHandler` 是否正常清除 EPWM2/EPWM3 的上下比较中断标志 | EPWM2/EPWM3 上下比较中断标志正常清除，系统不死机 | | ☐ |

## 四、蓝牙通信测试

| 编号 | 测试项 | 测试步骤 | 预期结果 | 实际结果 | 状态 |
|------|--------|----------|----------|----------|------|
| B-01 | 蓝牙初始化 | 上电后检查P34/P35 | 电源关闭时为GPIO浮空输入 | | ☐ |
| B-02 | 蓝牙上电 | 按电源键 `PowerKeyCnt=1` | P34切TXD1，P35切RXD1，蓝牙电源开启 | | ☐ |
| B-03 | 蓝牙连接状态 | 调用 `Bluetooth_GetConnect()` | 已连接返回1，未连接返回0 | | ☐ |

| B-05 | 帧头检测 | 发送 `55 AA` 开头的数据 | 解码器step从0进入1再进入2 | | ☐ |
| B-06 | 帧超时 | 发送 `55` 后等待>2秒 | 超时标志置位，不完整帧被丢弃 | | ☐ |
